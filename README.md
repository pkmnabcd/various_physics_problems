# Purpose of this Repo
This repo is where I document physics problems that I solve as I continue to practice and learn, even after I finish the required physics classes for my undergrad.
It contains my solutions to textbook problems, but not the problems themselves.
Although questions may overlap, this repo contains no solutions to my homework problem sets from my university studies.

# How to View Properly
I have written all of my solutions mainly in LaTeX, so to see a nice rendering of my work, you'll need a LaTeX compiler to PDF.

# Referenced Textbooks
This repo contains solutions to problems from the following textbooks.
* "Essential University Physics Fourth Edition Volume One" - Richard Wolfson
* "An Introduction to Thermal Physics" - Daniel V. Schroeder
